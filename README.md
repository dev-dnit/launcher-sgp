# SGP Launcher (CLI)

Servidor para automatizar a execução e atualização da aplicação SGP.

## Funcionalidades

- Verificação e download da versão mais recente do SGP.
- Encerramento do processo antigo antes de iniciar o novo.
- Injeção de variáveis de ambiente a partir dum variaveis.json.

## Instalação e Configuração

1. Clone este repositório e rode `npm install` para baixar as dependências.

## Como Usar

Para executar o launcher, simplesmente rode o comando no terminal:

```bash
npm run start
